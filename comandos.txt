Comandos no bash do git

git init: validar e iniciar uma pasta de versionamento.

git add nome_do_arq: leva para uma área de antes do commit.

git config --global user.name "Nome Sobrenome": nome do autor geral do de todos os projetos.

git --config user.email "endereçodeemail": define o endereço da conta.

git add --all: leva todos os arquivos para a área antes do commit.

git commit -m "nome do que foi feito na versão": o versionamento foi feito.  
 git push "da_origem_para" "nome_do_ramo".
