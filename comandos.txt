Comandos para o bash do git ou para prompt de comandos do Windows:

git init: validar e iniciar um repositório de versionamento.

git add nome_do_arq: leva para uma área de antes do commit.

git config --global user.name "Nome_Sobrenome": nome do autor geral do de todos os projetos.

git config user.email "endereço_de_email": define o endereço da conta.

git remote add "nome_de_origem" "endereço_do_repositório_remoto": nomeia e adiciona o endereço do repositório remoto

git add --all: leva todos os arquivos para a área antes do commit.

git commit -m "nome do que foi feito na versão": o versionamento foi feito.  

git branch -M "nome_do_ramo": cria ou move-se para o ramo

git push "nome_origem" "nome_do_ramo": atualiza o repositório remoto a partir do repositório local.

git pull "nome_de_origem"  "nome_do_ramo": atualiza o repositório local partir do repositório remoto.
